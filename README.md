Employee Full-Stack Application (Spring Boot + React.js)

A complete full-stack employee management system built using Spring Boot REST API (Backend) and React.js (Frontend).
This project demonstrates real-world full-stack development skills including:

REST API design

CRUD operations

React component-based UI

Clean architecture

State management

JSON API integration

Full developer workflow (backend → API → frontend UI)

This project is recruiter-friendly and suitable for placements, internships, and full-stack developer roles.
1. Backend — Spring Boot

RESTful API using Spring Boot

CRUD operations (Create, Read, Update, Delete)

Layered architecture (Controller → Service → Repository → Entity)

Easily extendable

Works with React frontend

Runs on port 8080
2. Frontend — React.js

Responsive UI

Add, Update, Delete employees

List employee records

API integration using Axios

Modern design (CSS based)

Clean components

Runs on port 3000

3. Folder Structure
employee-fullstack-app/
   1. backend  → Spring Boot API
   2. frontend → React.js UI
